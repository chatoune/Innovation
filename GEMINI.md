# Gemini Project Rules

These rules are foundational mandates for this project and take absolute precedence over general workflows.

## Data Privacy & Security
- **No Data Extraction:** Gemini is strictly forbidden from extracting, viewing, or printing the actual content/rows of any database.
- **Metadata Only:** Gemini may only inspect database metadata, such as table names, column names, and data types.

## Development Workflow
- **Validation Before Modification:** Gemini must never directly modify source code without first proposing the specific changes and obtaining explicit validation from the user.
- **Code Standards:** Adhere to the existing service/repository pattern and UI structure (PyQt/PySide).
